# Extension Incompatibilities

- `twxs.cmake`:<br>
  `CMake` semantic token highlighting depends on `josetr.cmake-language-support-vscode`, **not** `twxs.cmake`. This because the latter has a *worse* and sometimes *broken* tokenization.
